### Task 3: Add DTD skip

**Files:**
- Modify: `src/xml.sls` — add `parse-dtd` inside `xml-load`, modify `#\!` case in `parse-struct`

**Interfaces:**
- Produces: `parse-dtd` — internal helper, consumes chars via `next`, returns `'dtd` or raises error
- Modifies: `#\!` branch in `parse-struct` — detects DOCTYPE keyword and routes to `parse-dtd`

- [ ] **Step 1: Add `parse-dtd` internal function**

Insert after `parse-decl` definition and before `end-single-element`:

```scheme
      (define (parse-dtd)
        (let loop ([depth 0])
          (let ([ch (next)])
            (when (eof-object? ch)
              (error #f
                     "Parser Error: DTD not terminated"))
            (case ch
              [#\[ (loop (+ depth 1))]
              [#\] (loop (- depth 1))]
              [#\> (if (zero? depth)
                       'dtd
                       (loop depth))]
              [else (loop depth)]))))
```

- [ ] **Step 2: Add DOCTYPE detection in `parse-struct` `#\!` case**

Replace the `#\!` case in `parse-struct` (after the `#\?` case) with:

```scheme
            [#\! (let ([ch (next)])
                   (if (char=? ch #\-)
                       (parse-comment)
                       (let ([keyword (get-keyword (list ch))])
                         (if (string-ci=? keyword "DOCTYPE")
                             (parse-dtd)
                             (begin
                               (for-each (lambda (c)
                                           (push! ch-buffer c))
                                         (string->list keyword))
                               (parse-decl))))))]
```

- [ ] **Step 3: Add `get-keyword` helper inside `xml-load`**

Insert after `rlist->string` definition:

```scheme
      (define (get-keyword ch-rl)

              (if (or (eof-object? ch)
                      (not (char-alphabetic? ch)))
                  (begin
                    (push! ch-buffer ch)
                    (rlist->string ch-rl))
                  (get-keyword (cons ch ch-rl))))))
```
